# WordPress eCommerce Application This project is a WordPress-based eCommerce application powered by WooCommerce. It includes features for product management, customer interaction, and seamless shopping experiences. ## Table of Contents 1. [Requirements](#requirements) 2. [Installation](#installation) 3. [Configuration](#configuration) 4. [Features](#features) 5. [Development Workflow](#development-workflow) 6. [Troubleshooting](#troubleshooting) ## Requirements - **Web Server:** Apache/Nginx - **Database:** MySQL 5.6+ or MariaDB 10.0+ - **PHP:** Version 7.4 or later - **WordPress:** Version 6.0 or later - **WooCommerce:** Version 7.0 or later - **Browser:** Modern web browser (e.g., Chrome, Firefox, Edge) ## Installation ### Step 1: Setup Local Server 1. Install a local server environment (e.g., XAMPP, WAMP, or MAMP). 2. Enable the Apache `mod_rewrite` module. ### Step 2: Install WordPress 1. Download WordPress from [wordpress.org](https://wordpress.org/). 2. Extract the files into your server's web directory (e.g., `htdocs/ecommerce/`). 3. Create a new MySQL database for the application. 4. Configure the `wp-config.php` file with database details. ### Step 3: Install WooCommerce 1. Log in to the WordPress admin dashboard. 2. Navigate to **Plugins > Add New**. 3. Search for "WooCommerce" and install it. 4. Activate the plugin and follow the setup wizard. ### Step 4: Install Required Plugins Install any additional plugins required for the project. ### Step 5: Theme Installation 1. Install the Woostify theme (or your chosen WooCommerce-compatible theme). 2. Activate the theme and configure its settings via **Appearance > Customize**. ## Configuration ### Permalinks 1. Navigate to **Settings > Permalinks**. 2. Select the "Post name" structure. 3. Save changes to flush rewrite rules. ### WooCommerce Settings 1. Go to **WooCommerce > Settings** and configure: - **General:** Store address, currency options. - **Products:** Measurement units, review settings. - **Payments:** Enable and configure payment gateways. - **Shipping:** Define shipping zones and methods. ### Product Management 1. Add products via **Products > Add New**. 2. Include essential details: - Title, description, and short description. - Price and sale price. - Product image and gallery. - Categories and tags. ## Features - **Dynamic Product Catalog:** Add, edit, and manage products easily. - **User Authentication:** Customer registration and login. - **Payment Integration:** Secure and flexible payment gateways. - **Shipping Options:** Configurable shipping methods and zones. - **Customizable UI:** Theme customization for branding. ## Development Workflow ### Local Development - Use a local environment for testing changes. - Regularly update WordPress core, themes, and plugins. ### Version Control - Use Git for version control. Add `.gitignore` to exclude unnecessary files. ### Testing - Test functionalities like checkout, product updates, and user registration. ## Troubleshooting ### Common Issues and Fixes 1. **Permalink Issues:** - Go to **Settings > Permalinks** and save changes to flush rewrite rules. - Ensure `.htaccess` file is correctly configured. 2. **Broken Links:** - Verify WooCommerce and WordPress settings for URL structures. - Check custom menu items and update their links. 3. **Plugin Conflicts:** - Deactivate all plugins and reactivate them one by one to identify conflicts. 4. **Theme Issues:** - Switch to a default theme (e.g., Twenty Twenty-Three) to check for theme-related errors. <<<<<<< HEAD ### Identifiants administrateur principal -najikamal100@gmail.com mdp:90moXo4j%#nHdrI5M9 ======= ### Identifiants administrateur principal -najikamal100@gmail.com mdp:90moXo4j%#nHdrI5M9 >>>>>>> bfe72bf36be10ad1abcd0d77775effb705aa5bb2
